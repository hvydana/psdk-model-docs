# GStreamer

## Overview
Multimedia pipeline framework for streaming and processing

## Category
Media Pipeline Framework

## Components
- Gst-ONNX-RT: ONNX Runtime plugin
- GstHIP: HIP acceleration plugin

## Use Cases
- Video processing pipelines
- Real-time inference
- Camera streaming
- Industrial vision

## Integration
- ONNX-RT (Gst-ONNX-RT)
- HIP acceleration (GstHIP)
- CVML SDK
- ROS2

## Applications
- Veo Follow-Cam
- Industrial monitoring
- Robotics perception

## Description
Media pipeline framework providing streaming and processing infrastructure for video-based AI applications in industrial and robotics systems.
