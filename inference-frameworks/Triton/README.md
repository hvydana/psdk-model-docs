# Triton

## Overview
GPU programming and GEMM tuning framework

## Category
Compute Optimization

## Use Cases
- GEMM kernel optimization
- Custom GPU kernels
- Performance tuning
- Operator fusion

## Features
- Python-based GPU kernel programming
- Automatic optimization
- GEMM tuning for matrix operations
- ROCm/HIP integration

## Integration
- vLLM
- ROCm/HIP Stack
- Custom operator development

## Hardware Support
- AMD GPUs (ROCm)
- NVIDIA GPUs (CUDA)

## Description
GPU programming framework for optimizing matrix operations and custom kernels, used for tuning inference performance in the Physical AI SDK.
