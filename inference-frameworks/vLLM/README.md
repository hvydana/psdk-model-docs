# vLLM

## Overview
High-throughput and memory-efficient LLM inference engine

## Category
LLM Inference Framework

## Use Cases
- Large Language Models
- Vision-Language Models
- Multi-modal models
- Decoder models

## Supported Models
- LLaMA 3.2 Vision 11B
- DeepSeek-R1
- Qwen3 family
- OpenVLA
- Med-PaLM

## Key Features
- PagedAttention for efficient memory management
- Continuous batching
- Optimized for decoder models
- SpecDecode support (Eagle3 Drafter)

## Integration
- RyzenAI Stack
- ROCm/HIP Stack
- Lemonade orchestration

## Hardware Acceleration
- GPU-optimized (ROCm)
- APU support

## Description
Primary inference runtime for large language models and vision-language models in the Physical AI SDK, optimized for decoder-based architectures.
