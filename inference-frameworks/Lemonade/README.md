# Lemonade

## Overview
Orchestration framework for AI inference pipelines

## Category
Inference Orchestration

## Use Cases
- Multi-model pipeline orchestration
- Heterogeneous execution management
- Model scheduling
- Resource allocation

## Features
- Multi-runtime coordination (ONNX-RT, vLLM, etc.)
- APU resource management
- Pipeline optimization
- Workload scheduling

## Integration
- ONNX-RT
- vLLM
- GStreamer
- RyzenAI Stack

## Description
Orchestration layer for managing complex AI inference pipelines across heterogeneous hardware and multiple runtimes in the Physical AI SDK.
