# SpecDecode (vLLM Eagle3 Drafter)

## Overview
Speculative decoding for accelerated LLM inference

## Category
Inference Optimization

## Use Cases
- LLM inference acceleration
- Latency reduction
- Token generation speedup

## Technology
- Eagle3 Drafter model
- Speculative parallel decoding
- Draft-then-verify approach

## Supported Models
- DeepSeek-R1
- LLaMA family
- General decoder models

## Integration
- vLLM runtime
- Part of inference optimization pipeline

## Description
Speculative decoding technique using Eagle3 drafter model to accelerate LLM token generation by predicting multiple tokens in parallel.
