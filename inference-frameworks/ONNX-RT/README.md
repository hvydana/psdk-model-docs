# ONNX Runtime (ONNX-RT)

## Overview
Cross-platform inference engine for ONNX models

## Category
Inference Framework

## Use Cases
- Computer Vision models
- Encoder models
- Edge deployment
- Cross-platform inference

## Supported Models
- MobileSAM, YoloV12, CrossFormer
- MobileNetV3, EfficientNetV2
- 3D medical imaging models
- Most CV and traditional ML models

## Integration
- GStreamer: Gst-ONNX-RT plugin
- RyzenAI Stack
- CVML SDK

## Hardware Acceleration
- CPU execution
- GPU execution (DirectML, CUDA)
- NPU acceleration (RyzenAI)

## Description
Primary inference runtime for computer vision and traditional models in the Physical AI SDK, providing optimized execution across CPU, GPU, and NPU.
